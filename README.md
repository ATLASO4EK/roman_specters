 # roman_specters
Для запуска пайплайна нужно выполнить 2 простых шага:
1) Необходимо установить все нужные библиотеки, сделать это можно командой ```pip install -r requirements.txt```
2) В консоль необходимо написать команду ```python pipline.py "<имя файла>"```
файл должен находится в папке с моделью и пайплайном
